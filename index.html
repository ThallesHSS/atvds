Tabelas de notas escolares

Instruções

Crie uma tabela HTML que represente o boletim de notas de uma turma escolar.
A tabela deve conter as seguintes caracteristicas

Instruções

  1. Cabeçalho da Tabela Com titulo usando ⁢tcaption>
  2. Duas linhas de cabeçalho usando ⁢thead>
  3. Corpo da tabela com dados de cinco alunos usando ⁢tbody>
  4. Rodapé com estatistica usando ⁢tfoot>
  5. Uso de ⁢th> para células de cabeçalho
  6. Uso de colspan para mesclar células horizontalmente
  7. Uso de rowspan para mesclar células verticalmente
  8. atributo scope para acessibilidade

Exercício prático

Crie sua própria tabela seguindo este modelo, mas com os seguintes dados:

Dados para usar

  • Turma: 2ºV03-IPI
  • Disciplinas: Matemática, Português, Geografia, Inglês
  • Alunos:
    • Gabriel Zancanelli: 6,9 - 4,5 - 10,0 - 10,0
    • Juscelino Kubitscheck: 5,0 - 9,5 - 2,0 - 7,0
    • Levandowski da Silva: 10,0 - 7,5 - 8,3 - 9,4
    • Cristiano Ronaldo: 2,5 - 7,5 - 9,3 - 8,6
    • Karoline de Mello Ladislau Stein Filho: 3,5 - 5,0 - 2,0 - 7,9

Calcule as Médias

Para calcular a média geral de cada aluno some as quatro notas e divida as por quatro

Para a Situação:

  • Aprovado: Média >= 7,0
  • Recuperação: Média entre 5 e 6,9
  • Reprovado: Média < 5,0

Dica para resolver o exercício

  1. Começe pela estrutura básica da tabela com ⁢table>
  2. Adicione ⁢caption> com o título
  3. Crie o ⁢thead> com as duas linhas de cabeçalho
  4. Use colspan="4" para a célula "Disciplinas"
  5. Use rowspan="2" para as células "Alunos, Média Geral e Situação"
  6. Preencha com ⁢tbody> os dados dos alunos
  7. Calcule as médias e defina a situação de cada aluno
  8. Adicione o ⁢tfoot> com as estatísticas finais
  9. Use scope="col" para cabeçalhos de coluna e scope="row" para cabeçalhos de linha

Boa sorte com o exercício! Lembre-se de testar sua tabela em um navegador para verificar se está funcionando corretamente
